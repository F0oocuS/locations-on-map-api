# Location on map API

Про проект:
- використовується node.js + express.
- бази даних немає.

Робота з проектом:
- npm install - завантаження залежностей.
- npm run dev - запуск проекту на 3000 порті.

Структура проекту:
- точка входу app.js.
- src/controllers - папка з контролерами.
- src/models - папка з моделями. В даному випадку модель слугує ще й місцем зберігання даних.
- src/routes - папка з роутами.
- scr/scripts - папка для зберігань усіх скриптів, що використовувались під час розробки для генерації тестових даних та завантаження тайлів карти.
